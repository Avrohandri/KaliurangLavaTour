# ⛩ Jeep Gerbang Kaliurang — Lava Tour Merapi

Website profil dan pemesanan untuk jasa **Jeep Lava Tour Merapi** yang berlokasi di Kaliurang, Yogyakarta.

## Tentang

Website ini dibuat sebagai landing page untuk memperkenalkan layanan wisata jeep off-road di kawasan lereng Gunung Merapi. Pengunjung dapat melihat paket tur yang tersedia beserta harganya, mengenal fasilitas yang disediakan, dan langsung memesan melalui WhatsApp.

## Fitur

- Informasi paket tur (Kaliurang Explore, Short Trip, Short Plus, Sunrise Trip, Medium Trip, Long Trip)
- Galeri foto destinasi wisata
- Informasi lokasi basecamp dan kontak
- Tombol pesan langsung via WhatsApp

## Teknologi

- HTML, CSS (Tailwind CSS)
- Deployed di [Vercel](https://kaliurang-lava-tour.vercel.app/)
